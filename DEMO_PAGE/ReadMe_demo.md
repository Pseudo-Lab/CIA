# 데모 페이지 템플릿

1. 아래 명령어를 통해 통합 데모페이지 실행
- python app_template.py

2. pages 폴더 내 각자의 데모 파일을 만들어 app_template.py 에 삽입할 예정
- pages 내 파일을 보시면서 전체 인과추론 실험 및 분석 내용 정리 + 시연용 탭 개발 (사용자 input -> (추천) 결과 시각화)
